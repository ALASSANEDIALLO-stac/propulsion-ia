Propulsion IA 🎬
Propulsion IA est une application web puissante conçue pour tous les créateurs de contenu qui souhaitent gagner du temps et optimiser leur présence sur TikTok. Cet outil vous permet de centraliser, planifier et gérer vos publications de manière professionnelle.

✨ Fonctionnalités
✅ Planification Complète : Gérez vos dates et heures de publication pour ne jamais manquer un créneau.

✅ Gestion Multimédia : Ajoutez vos vidéos originales et personnalisez vos vignettes (miniatures).

✅ Optimisation du Contenu : Préparez vos descriptions, légendes et hashtags à l'avance.

✅ Tableau de Bord : Suivez l'état de vos publications (Planifié, Brouillon, Publié) avec des statistiques claires.

✅ Interface Moderne : Une gestion fluide et rapide pour rester concentré sur la création.

🚀 Installation et démarrage
Windows
Double-clique sur le fichier start.bat

Mac / Linux
Bash
chmod +x start.sh
./start.sh
Installation Manuelle
Installez les dépendances :

Bash
pip install -r requirements.txt
Lancez l'application :

Bash
python app.py
Ouvrez votre navigateur sur : http://localhost:5000

📂 Structure du projet
propulsion-ia/
├── app.py              # Serveur Python (Moteur de l'application)
├── .env                # Vos clés secrètes (TikTok API)
├── requirements.txt    # Bibliothèques nécessaires
├── start.bat           # Lancement rapide Windows
├── start.sh            # Lancement rapide Mac/Linux
├── templates/
│   └── index.html      # Interface utilisateur
├── uploads/            # Stockage de vos vidéos et images
└── data/
    └── posts.json      # Sauvegarde de vos planifications
⚖️ Note Importante
Propulsion IA utilise l'API officielle de TikTok pour garantir une publication sécurisée. Les utilisateurs sont responsables du respect des droits d'auteur sur les contenus publiés. L'outil est conçu pour automatiser le flux de travail des créateurs originaux.
